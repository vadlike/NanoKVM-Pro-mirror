## NanoKVM Mirror

Portable Windows viewer for the local NanoKVM LCD over SSH.

### Included

- standalone `NanoKVM-Mirror.exe`
- Python source `kvm-screen-mirror.py`
- demo `gif.gif`
- `README.md`
- optional launchers `.cmd` and `.vbs`
- `kvm-screen-mirror.example.json`

### Features

- mirrors the NanoKVM screen on Windows
- click to tap and drag to swipe
- knob controls and extra helper buttons
- portable config saved beside the app
- custom frameless UI

### Notes

- `kvm-screen-mirror.json` is not included in release assets
- create your own config from `kvm-screen-mirror.example.json` if needed
